## Features
- 3D third-person and first-person camera modes
- Player movement and rotation
- Bullet shooting mechanics
- Enemy AI that follows the player
- Bullet–enemy collision detection
- Player–enemy collision handling
- Score tracking system
- Life system with game-over condition
- Cheat mode with auto-aim and auto-shoot
- Dynamic camera control
- Restartable gameplay

## Game Mechanics
- Enemies continuously move toward the player
- Enemy size dynamically oscillates
- Bullets have limited lifespan
- Missing too many bullets ends the game
- Player loses life on enemy contact
- Game ends when life reaches zero

## Controls
- **W / S** – Move forward / backward
- **A / D** – Rotate player left / right
- **Left Mouse Button** – Shoot bullets
- **Right Mouse Button** – Toggle first-person camera
- **Arrow Keys** – Move third-person camera
- **C** – Toggle cheat mode
- **V** – Toggle cheat camera view
- **R** – Restart game (after game over)
